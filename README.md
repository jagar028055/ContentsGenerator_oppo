## 🛠️ 使い方

### Webインターフェース
`index.html` をブラウザで開くと、プロンプトと生成画像を記録・管理できるインターフェースが使えます。

### 主な機能
- プロンプトと生成画像の記録
- プロンプトでの検索・フィルタリング
- プロンプトのクリップボードコピー
- データのエクスポート/インポート（JSON形式）
- 画像のダウンロード
- ローカルストレージでのデータ保存
